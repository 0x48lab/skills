概要

SkillsプラグインはクラシックMMORPGにインスパイアされたスキルシステムをマインクラフトに導入します。プレイヤーは行動によってスキルを向上させ、スキル合計上限により特化か万能かの選択を迫られます。

主な機能

  • 40スキル - 戦闘、魔法、生産、採集、シーフ、テイム、サバイバルの各カテゴリ
  • スキルキャップ - 合計700ポイント、各スキル最大100ポイント
  • 使用による成長 - スキルは使用することで上昇、難易度が成長率に影響
  • ステータスシステム - 関連スキルからSTR、DEX、INTを計算
  • 品質クラフト - 生産スキルによりアイテム品質が変動
  • 触媒魔法 - 触媒を消費する8サークルの魔法システム

ステータスシステム

3つの内部ステータスが関連スキルから計算されます：

ステータス 効果 関連スキル
STR +HP、+採掘・伐採速度 Swordsmanship, Axe, Mace Fighting, Wrestling, Tactics, Anatomy, Mining, Lumberjacking, Blacksmithy, Craftsmanship, Cooking, Heat/Cold Resistance, Endurance
DEX +攻撃速度、+移動速度 Archery, Throwing, Parrying, Focus, Hiding, Stealth, Snooping, Stealing, Poisoning, Fishing, Tinkering, Athletics, Swimming
INT -マナ消費、+詠唱成功率 Magery, Evaluating Intelligence, Meditation, Resisting Spells, Inscription, Alchemy, Bowcraft, Animal Lore

ステータス計算

ステータス = 関連スキル合計 / 関連スキル数

各ステータスは0〜100で、関連スキルの平均値となります。

ステータス効果

ステータス 効果計算式
STR 最大HP = 100 + STR、採掘・伐採速度 +STR/10%
DEX 攻撃速度 +DEX/2%、移動速度 +DEX/10%
INT マナ消費 -INT/2%、詠唱成功率 +INT/5%

スキル

スキル上昇システム

上昇確率 = (100 - スキル値) / 10 × 難易度補正

上昇量：判定成功時 0.1ポイント

難易度補正

条件 補正
難易度 > スキル + 20 ×0.5（難しすぎる）
難易度がスキル±20以内 ×1.0（最適）
難易度 < スキル - 20 ×0.2（簡単すぎる）

戦闘システム

ダメージ計算式

基礎ダメージ = 武器の基礎攻撃力（設定ファイルから）
STRボーナス = 1 + (STR / 200)     // 最大 +50%
Tacticsボーナス = Tactics / 100    // 0.1 〜 1.0
Anatomyボーナス = 0.5 + (Anatomy / 100)  // 0.6 〜 1.5
クリティカル = Anatomy / 2 % の確率で2倍ダメージ

最終 = 基礎 × STR × Tactics × Anatomy × 品質 × エンチャント

命中率

命中率 = 武器スキル - (対象防御値 / 2) + 50

最小：5%、最大：95%

パリィ

装備 ブロック確率 ダメージ軽減
盾 Parrying × 0.5%（最大50%） 50%
武器のみ Parrying × 0.25%（最大25%） 50%

魔法システム

詠唱条件

  • スペルブック必須 - その魔法が記されたスペルブックを所持
  • 触媒 - 魔法ごとに指定のマインクラフトアイテムを消費
  • マナ - 空腹ゲージを使用
  • ターゲット - 多くの魔法は詠唱後にターゲットモードに入る

詠唱成功率

成功率 = (Magery - サークル × 10) + 50 + (INT / 5)

最小：5%、最大：100%

生産システム

生産スキルはアイテムの品質と特殊効果に影響します。スキルが高いほど良い品質と強いボーナスが得られます。

品質システム

品質 必要スキル 攻撃力/防御力 耐久度
低品質（LQ）0-49-15%-25%
通常品質（NQ）50-69+0%+0%
高品質（HQ）70-89+15%+30%
最高品質（EX）90-100+25%+50%

Cookingスキル

調理した食べ物にCookingスキルに基づいた回復ボーナスが付与されます。

回復ボーナス = Cookingスキル / 4 + 品質補正

LQ: -15%、NQ: +0%、HQ: +15%、EX: +25%

採集システム

採集スキルは資源収集時にボーナスを提供します。スキルが高いほど、採集速度が上がり、ドロップが良くなり、ツールの耐久消費が軽減されます。

Miningスキル

Miningスキルは鉱石採掘（ツルハシ）とソフトブロックの掘削（シャベル）の両方に適用されます。

速度ボーナス（採掘速度上昇効果）

鉱石を採掘したり、ソフトブロックを掘り始めると、スキルに応じた採掘速度上昇効果が適用されます。

Miningスキル 速度ボーナス 採掘速度上昇レベル
0-190-9%なし
20-3910-19%採掘速度上昇 I
40-7920-39%採掘速度上昇 I
80-10040-50%採掘速度上昇 II

耐久消費軽減

軽減確率 = スキル × 0.9%（GM = 100%）

スキル100（グランドマスター）では、ツールの耐久度が減らなくなります。

Lumberjackingスキル

斧で原木を伐採する際、スキルが高いほど効率が上がります。

速度ボーナス（採掘速度上昇効果）

速度ボーナス = Lumberjackingスキル / 2（最大 +50%）

耐久消費軽減

軽減確率 = スキル × 0.9%（GM = 100%）

Fishingスキル

Fishingスキルは釣り待ち時間を短縮し、宝物確率を向上させます。

待ち時間短縮

待ち時間短縮 = スキル / 5（最大 -20%）

宝物確率

宝物ボーナス = スキル / 4（最大 +25%）

修理・修繕システム

金床修理

金床でアイテムを修理すると、最大耐久度が減少するリアルな摩耗システムです。

最大耐久度減少 = (110 - Blacksmithyスキル) / 10
Blacksmithyスキル 減少量

修繕の成功率はアイテム種別に応じた生産スキルに依存します。

修繕成功率 = min(100, スキル × 100 / 60)
