概要

SkillsプラグインはクラシックMMORPGにインスパイアされたスキルシステムをマインクラフトに導入します。プレイヤーは行動によってスキルを向上させ、スキル合計上限により特化か万能かの選択を迫られます。

主な機能

  • 38スキル - 戦闘、魔法、生産、採集、シーフ、テイム、サバイバルの各カテゴリ
  • スキルキャップ - 合計700ポイント、各スキル最大100ポイント
  • 使用による成長 - スキルは使用することで上昇、難易度が成長率に影響
  • ステータスシステム - 関連スキルからSTR、DEX、INTを計算
  • 品質クラフト - 生産スキルによりアイテム品質が変動
  • 触媒魔法 - 触媒を消費する8サークルの魔法システム

ステータスシステム

3つの内部ステータスが関連スキルから計算されます：

ステータス 効果 関連スキル
STR +HP、+採掘・伐採速度 Swordsmanship, Axe, Mace Fighting, Wrestling, Tactics, Anatomy, Mining, Lumberjacking, Blacksmithy, Craftsmanship, Cooking, Heat/Cold Resistance, Endurance
DEX +攻撃速度、+移動速度 Archery, Throwing, Parrying, Focus, Hiding, Stealth, Snooping, Stealing, Poisoning, Fishing, Athletics, Swimming
INT -マナ消費、+詠唱成功率 Magery, Evaluating Intelligence, Meditation, Resisting Spells, Inscription, Alchemy, Animal Lore
複合 個別スキル参照 Craftsmanship（STR 50% + DEX 50%）、Animal Taming、Veterinary、Detecting Hidden、Arms Lore

ステータス計算

ステータス = 関連スキル合計 / 関連スキル数

各ステータスは0〜100で、関連スキルの平均値となります。

ステータス効果

ステータス 効果計算式
STR 最大HP = 100 + STR、採掘・伐採速度 +STR/10%
DEX 攻撃速度 +DEX/2%、移動速度 +DEX/10%
INT マナ消費 -INT/2%、詠唱成功率 +INT/5%

スキル

スキル上昇システム

上昇確率 = (100 - スキル値) / 10 × 難易度補正

上昇量：判定成功時 0.1ポイント

難易度補正

条件 補正
難易度 > スキル + 20 ×0.5（難しすぎる）
難易度がスキル±20以内 ×1.0（最適）
難易度 < スキル - 20 ×0.2（簡単すぎる）

サバイバル系スキル

※環境ダメージはバニラと同等の致死性を維持。STRによるHP増加の恩恵は20%に制限（80%はダメージスケールに反映）。スキル100で完全免疫。

ダメージ倍率 = 0.5 × STRスケール × (1 - スキル/100)
STRスケール = 1.0 + ((maxHP/100 - 1) × 0.8)

例: STR 25 (maxHP 125), スキル 0
  STRスケール = 1.0 + (0.25 × 0.8) = 1.2
  ダメージ倍率 = 0.5 × 1.2 × 1.0 = 0.6
  → バニラ致死ダメージでほぼ死亡（STRの恩恵は5%程度）

戦闘システム

ダメージ計算式

基礎ダメージ = 武器の基礎攻撃力（設定ファイルから）
STRボーナス = 1 + (STR / 200)     // 最大 +50%
Tacticsボーナス = Tactics / 100    // 0.1 〜 1.0
Anatomyボーナス = 0.5 + (Anatomy / 100)  // 0.6 〜 1.5
クリティカル = Anatomy / 2 % の確率で2倍ダメージ

最終 = 基礎 × STR × Tactics × Anatomy × 品質 × エンチャント

命中率

命中率 = 武器スキル - (対象防御値 / 2) + 50

最小：5%、最大：95%

パリィ

装備 ブロック確率 ダメージ軽減
盾 Parrying × 0.5%（最大50%） 50%
武器のみ Parrying × 0.25%（最大25%） 50%

魔法システム

詠唱条件

  • スペルブック必須 - その魔法が記されたスペルブックを所持
  • 触媒 - 魔法ごとに指定のマインクラフトアイテムを消費
  • マナ - 空腹ゲージを使用
  • ターゲット - 多くの魔法は詠唱後にターゲットモードに入る

詠唱成功率

成功率 = (Magery - サークル × 10) + 50 + (INT / 5)

最小：5%、最大：100%

ルーン

ルーンはMark、Recall、Gate Travel魔法に必要なアイテムです。空のルーンに場所を記録してから使用します。

ルーンの入手方法

インベントリ内でTeleportスクロールとアメジストの欠片を合成：

  1. Teleportスクロールをカーソルで持つ
  2. インベントリ内のアメジストの欠片をクリック
  3. 空のルーンが作成される（逆の操作でも可）

スキル不要、成功率100%

Teleportスクロールの入手先： Mobドロップ（ウィッチ等）、End City宝箱、トライアルチャンバー宝物庫、自作（/scribe teleport）

ルーンの使い方

手順 操作
1空のルーンをメインハンドに持つ
2/cast mark で場所を記録
3記録済みルーンでRecallまたはGate Travelを詠唱

ルーンの書

複数のルーンを保存できる魔法の本。チェストのようにルーンを自由に出し入れでき、GUIから直接RecallやGate Travelを詠唱できます。

入手方法

  • 司書の村人（Lv.3）から購入（40〜60エメラルド）

使い方

操作 説明
右クリック（本）ルーンの書GUIを開く
ルーン登録マーク済みルーンのみ登録可能（最大16個）

GUI操作

操作 説明

※ 右クリック系以外は通常のチェストと同じ動作です。通常詠唱と同じ触媒・マナ・詠唱時間が必要です。

採集システム

採集スキルは資源収集時にボーナスを提供します。スキルが高いほど、採集速度が上がり、ドロップが良くなり、ツールの耐久消費が軽減されます。

Miningスキル

Miningスキルは鉱石採掘（ツルハシ）とソフトブロックの掘削（シャベル）の両方に適用されます。

速度ボーナス（採掘速度上昇効果）

鉱石を採掘したり、ソフトブロックを掘り始めると、スキルに応じた採掘速度上昇効果が適用されます。

Lumberjackingスキル

斧で原木を伐採する際、スキルが高いほど効率が上がります。高スキルになると隣接する原木を連鎖して伐採できます。

速度ボーナス（採掘速度上昇効果）

速度ボーナス = Lumberjackingスキル / 10（最大 +10%）

連鎖伐採のルール

  • 対象：原木のみ（葉は破壊しない）
  • 範囲：上方向のみ（上、斜め上の9方向）※下方向には連鎖しない
  • 耐久度：最初の1ブロック分のみ消費
  • スタミナ：消費なし
  • 別種の木：連鎖しない（オーク→オークのみ）
  • 斧の要件：不要（素手でも連鎖する）
  • ドロップ：各原木の位置にドロップし、自然落下する
  • 処理方式：分散処理（数tickに分けて破壊、サーバー負荷軽減）

  • 真ん中を切る：切った位置から上だけが伐採される（下は残る）
  • 上の方を切る：その部分から上だけが伐採される

Fishingスキル

Fishingスキルは自動釣りを可能にします。魚がかかると、スキルに応じた確率で自動釣り上げと再キャストが行われます。

自動釣りシステム

  • 手動キャスト：プレイヤーが手動で釣り竿をキャストして開始
  • 自動釣り上げ：魚がかかると、スキル値%の確率で自動的に釣り上げ
  • 自動再キャスト：自動釣り上げ成功後、自動的に再キャスト
  • ループ：釣り竿が壊れるかプレイヤーが手動で止めるまで継続

自動釣り上げ確率

スキルレベル 確率 説明
0-19なし手動のみ（バニラ動作）
20-99スキル値%確率で自動釣り上げ

備考

自動釣りはボーナス機能です。自動釣り上げが失敗しても、手動で釣り上げれば魚は釣れます（バニラ動作）。時間内に反応しなかった場合のみ魚が逃げます。

項目 説明

H（HP） 現在HP / 最大HP。最大HP = 100 + STR

STR/DEX/INT 各ステータス値とロックモード

ステータスロックモード

各ステータスの右側に表示されるアイコンは、そのステータスの変動モードを示します。ステータス合計上限（225）に達すると、あるステータスを上げるには別のステータスを下げる必要があります。

アイコン モード 説明
▲ UP（上昇） ステータスが上昇可能。合計上限に達している場合、他のステータス（DOWNまたはUPモード）を下げて上昇
▼ DOWN（下降） ステータスが下降のみ可能。他のステータスが上昇する際に優先的に下がる
■ LOCKED（固定） ステータスは一切変化しない。上昇も下降もしない

ロックモードの切り替え

/stats lock <str|dex|int>コマンドでロックモードを切り替えられます。

切り替え順序: UP → DOWN → LOCKED → UP...

使用例

  • 戦士ビルド：STRをUP、DEXをUP、INTをDOWNに設定 → INTを犠牲にSTR/DEXが上昇
  • 魔法使いビルド：INTをUP、STR/DEXをDOWNに設定 → STR/DEXを犠牲にINTが上昇
  • バランス型：全てUPに設定 → 均等に上昇（上限到達後は最も低いものが優先的に下がる）
  • 特定値を維持：維持したいステータスをLOCKEDに設定 → そのステータスは変化しない

他プラグインとの競合

設定ファイルのscoreboard.conflict_modeで他プラグインとのスコアボード競合時の動作を設定できます。

モード 説明
