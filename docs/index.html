Overview

The Skills plugin brings a classic MMORPG-inspired skill system to Minecraft. Players improve skills through use, with a total skill cap that forces specialization choices.

Key Features

  • 38 Skills across combat, magic, crafting, gathering, thief, taming, and survival categories
  • Skill Cap System - Total cap of 700 points, individual cap of 100 per skill
  • Use-Based Progression - Skills improve when used, with difficulty affecting gain rate
  • Stat System - STR, DEX, INT calculated from related skills
  • Quality Crafting - Item quality depends on crafting skill
  • Reagent Magic - Reagent-based spellcasting with 8 circles of magic

Stats System

Three internal stats are calculated from related skills:

Stat Effect Related Skills
STR +HP, +Mining/Lumber speed Swordsmanship, Axe, Mace Fighting, Wrestling, Tactics, Anatomy, Mining, Lumberjacking, Blacksmithy, Craftsmanship, Cooking, Heat/Cold Resistance, Endurance
DEX +Attack speed, +Movement speed Archery, Throwing, Parrying, Focus, Hiding, Stealth, Snooping, Stealing, Poisoning, Fishing, Athletics, Swimming
INT -Mana cost, +Cast success Magery, Evaluating Intelligence, Meditation, Resisting Spells, Inscription, Alchemy, Animal Lore
Mixed See individual skills Craftsmanship (STR 50% + DEX 50%), Animal Taming, Veterinary, Detecting Hidden, Arms Lore

Stat Calculation

Stat = (Sum of related skills) / (Number of related skills)

Each stat ranges from 0 to 100 based on the average of its related skills.

Stat Effects

Stat Effect Formula
STR Max HP = 100 + STR, Mining/Lumber speed +STR/10%
DEX Attack speed +DEX/2%, Movement speed +DEX/10%
INT Mana cost -INT/2%, Cast success +INT/5%

Skills

Skill Gain System

Gain Chance = (100 - Skill) / 10 * Difficulty Modifier

Gain Amount: 0.1 per successful check

Difficulty Modifiers

Condition Modifier
Difficulty > Skill + 20 x0.5 (too hard)
Difficulty within ±20 of Skill x1.0 (optimal)
Difficulty < Skill - 20 x0.2 (too easy)

Combat System

Damage Formula

Base Damage = Weapon base damage (from config)
STR Bonus = 1 + (STR / 200)     // Max +50%
Tactics Bonus = Tactics / 100    // 0.1 to 1.0
Anatomy Bonus = 0.5 + (Anatomy / 100)  // 0.6 to 1.5
Critical = Anatomy / 2 % chance for 2x damage

Final = Base × STR × Tactics × Anatomy × Quality × Enchant

Hit Chance

Hit Rate = Weapon Skill - (Target Defense / 2) + 50

Min: 5%, Max: 95%

Parrying

Equipment Block Chance Damage Reduction
Shield Parrying × 0.5% (max 50%) 50%
Weapon Only Parrying × 0.25% (max 25%) 50%

Magic System

Spellcasting

  • Spellbook Required - Must hold a spellbook with the spell
  • Reagents - Spells consume Minecraft items as reagents
  • Mana - Uses the hunger bar
  • Targeting - Most spells enter target mode after casting

Cast Success Rate

Success = (Magery - Circle × 10) + 50 + (INT / 5)

Min: 5%, Max: 100%

Stack Size Bonus

Production skills increase the maximum stack size of items from 64 to 99.

Max Stack Size = 64 + (Skill Sum / 400 × 35)

Contributing Skills: Craftsmanship + Blacksmithy + Cooking + Alchemy

Gathering System

Gathering skills provide bonuses when collecting resources. Higher skill means faster gathering, better drops, and reduced tool durability consumption.
